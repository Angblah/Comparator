What is a comparison?
A comparison is a table where you can compare products. Each comparison has a set of attributes going down the rows and a number of items going across the columns. Table contents can be edited by clicking within a cell and typing.
What is a template?
A template is a collection of attributes relevant to a product category. For example, a laptop template may have the attributes RAM, Price, Weight, Battery Life, etc. Templates can be used to quickly create a comparison comparing items in a specific category. To use a template, first select the "New Comparison" button on the bottom-right side of the dashboard. If you are not logged in, templates can be accessed in the "Start Comparing" button on the homepage. Click on each template name to see which attributes they have. To create a comparison using a template, simply click the "Start Comparing" button on a template dropdown. If you do not find a template that meets your needs, select "Empty Template" to start comparing with a blank comparison with no attributes.
How do I create a comparison?
Comparisons can be created by clicking the "New Comparison" button on the dashboard for logged in users, or the "Start Comparing" button on the homepage. A modal/popup should appear with a list of templates to use to create a comparison. If you do not find a template related to the product you wish to compare, select "Empty Template" to create a new empty comparison.
How do I create a template?
Templates can be created by clicking the save button on a comparison and selecting "template". This will create a new template with the comparison's attributes.
What features are limited for guests/why should I sign up?
While guests have access to most of the features of Comparator, certain web app functions are restricted to registered users due to implementation constraints. Guests can create and edit comparisons. However, they will need to manually save/remember the links to their comparisons to access them again, whereas registered users can access their comparisons on their dashboard.
How much storage space do I have?
Currently, image upload storage space is limited to 10 MB per account. For guests, this 10 MB limit includes all images uploaded by any guest user. Comparison/template storage space is not currently restricted.
